## Build & Run

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run lint     # Run ESLint
```

## Validation

After implementing, run:
- Build: `npm run build`
- Lint: `npm run lint`

## Codebase Patterns

- Icon constants: `lib/type-icons.tsx`
- Status icons: `lib/status-icons.tsx`
- Tag colors: `lib/tag-colors.ts`
- UI constants: `lib/ui-constants.ts`

Always search first, use existing patterns.
